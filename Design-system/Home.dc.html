// Quickstart

Live in three steps.

1

Create an API key

Sign up and generate a key from your dashboard in seconds.

2

Point your SDK at YouBox

Swap the base URL. Keep your existing OpenAI client untouched.

3

Pick any model

Pass a provider-prefixed slug and route to 300+ models instantly.

bash
# 1 — set your key
export YOUBOX_KEY="yb-prod-7f3a..."

# 2 — call any model, OpenAI-style
curl https://api.youbox.dev/v1/chat/completions \
  -H "Authorization: Bearer $YOUBOX_KEY" \
  -d '{
    "model": "google/gemini-3-pro",
    "messages": [{"role":"user","content":"Hi"}]
  }'
