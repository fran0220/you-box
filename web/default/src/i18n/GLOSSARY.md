## Do not translate

- Brand names: OpenAI, Claude, Gemini, Stripe, Creem, Discord, Telegram, WeChat, etc.
- URLs, email placeholders, JSON examples, model IDs
- Technical identifiers: OAuth, API, JSON, SMTP

## Style notes

- **zh**: Use concise product UI Chinese; avoid literal machine translation ("异常桶" → "事件")
- **ja**: Technical loanwords in katakana; polite です/ます form for user messages
- **ru**: Formal register (вы)
- **fr**: Standard French; technical terms may stay as cognates when natural
- **vi**: Standard Vietnamese; keep API/OAuth as loanwords when conventional
